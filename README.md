# PRODIGY_WD_03
A responsive calculator web application built using HTML, CSS, and JavaScript with arithmetic operations, keyboard support, error handling, and a modern interactive user interface.


#project name
Calculator Web Application

## Overview

This project is a modern and responsive Calculator Web Application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides a clean, user-friendly interface with keyboard support for a better user experience.

## Features

* Addition, Subtraction, Multiplication, and Division
* Percentage calculations
* Clear (C) button to reset calculations
* Backspace (⌫) button to delete the last character
* Keyboard support for numbers and operators
* Error handling for invalid expressions
* Responsive and modern UI design
* Smooth button hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Use

1. Open the application in a web browser.
2. Click the buttons or use your keyboard to enter numbers and operators.
3. Press "=" or Enter to calculate the result.
4. Use "C" to clear the display.
5. Use "⌫" or Backspace to delete the last entered character.
